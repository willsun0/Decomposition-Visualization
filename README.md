# Matrix Motion

An interactive, animated visual guide to principal component analysis (PCA), singular value decomposition (SVD), and QR decomposition.

## Deploy on GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, `app.js`, and this README to the repository root.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)`, then save.

No installation, build command, backend, or API key is required.

## Features

- Draggable PCA dataset with centering, covariance, principal axes, projection, and explained variance
- Step-by-step SVD animation showing input rotation, singular-value scaling, and output rotation
- Live matrix presets, singular values, rank, and condition number
- Draggable QR vectors with an animated Gram–Schmidt construction
- Responsive layout, keyboard navigation, and reduced-motion support

Use the top navigation to switch topics. Press Space to play or pause a walkthrough, and use the arrow keys to move between steps.
